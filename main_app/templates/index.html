Warhandler

Warhammer 40K is a tabletop miniatures game by Games Workshop, and it is one of the most popular army scale tabletop games in the world. As in other miniature wargames, players enact battles using miniature models of warriors and fighting vehicles. This website keeps track of all the different statistics of the units and allows you to build and set up your own units. This was made with React as a frontend framework and ExpressJS as the backend. To supplement this website, I am going to make a list making system that will allow a user to build their own army list.

A team of 3 members made this project:

  • Chris McClarin: Scrum and frontend developer
  • Andy Terwilliger: CSS Developer and Full Stack Developer
  • Kendlyn Marriott: Backend Developer and PitchDeck
